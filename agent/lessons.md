# Lessons Learned

> **上限**: 15エントリ / 120行。肥大化はコンテキストウィンドウを圧迫し全作業品質を下げる。
> **卒業**: 10件到達時に類似統合 →「カスタマイズファイルの選択基準」表に従い適切なカスタマイズファイルへの昇格を判断する。

## 2026-03-04: 卒業判断を todo に含めないと実行されない

- **パターン**: lessons.md が10件に達していたが、複数セッションにわたって卒業判断が実施されなかった
- **問題**: `manage_todo_list` に「卒業判断」タスクを含めないと、作業完了時に忘れる
- **ルール**: copilot-instructions.md のセッション開始手順に「末尾に卒業判断タスクを含める」を追加済み

## 2026-03-04: テストで設定値をハードコードしてはならない

- **パターン**: プラグイン TOML の `default = true` を `false` に変更しただけで CI が壊れた
- **問題**: テストが設定ファイルの具体的な値をべた書き（`assert_eq "PLUGIN_DEFAULT" "true"`）していた
- **ルール**: テストの期待値は設定ファイルから動的に読み取る。「コードの振る舞い」をテストし、「設定のスナップショット」を取らない
